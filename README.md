
# Projeto WEB - Site de Cinema

Durante a disciplina de Programação WEB, fomos desafiados para montar um site de cinema. Nele simulamos reservas e catálogo de filmes.
# O que foi feito?
Durante o desenvolvimento deste projeto, fizemos os seguintes materiais:

* Relatório técnico
* Documentação de requisitos e regras de negócio
* Normalização do banco de dados
* Desenvolvimento
* Cronograma
* Debates
* Slides de apresentação 

> Consulte todos os materiais na Wiki do nosso repositório!

# Diretórios

* **RavelCinema3.0/**: diretório onde está localizado o projeto finalizado integrado ao banco de dados.
* **model/**: diretório onde está localizado todos os arquivos do esquema Model do projeto.
* **database/**: diretório onde está localizado todos os arquivos relacionados ao banco de dados do projeto.
* **controller/**: diretório onde está localizado todos os arquivos do esquema Controller do projeto.
* **view/**: diretório onde está localizado todos os arquivos do esquema View do projeto.
* **config/**: diretório onde está localizado todos os arquivos relacionados as configurações de ambiente do projeto.
